Sales Analyzer is a Python-based data analysis project built using Pandas and Object-Oriented Programming (OOP). The project demonstrates how real-world sales data can be loaded, cleaned, filtered, analyzed, and merged with customer information to generate useful business insights.

The primary objective of this project is to practice and understand important Pandas concepts commonly used in data analysis, machine learning preprocessing, and business reporting.


## Features

### Data Loading

* Read sales data from CSV files
* Store data in a Pandas DataFrame

### Data Exploration

* View first few records
* Check dataset shape
* Display column names
* Inspect data types

### Data Cleaning

* Detect missing values

### Data Filtering

* Filter sales by city
* Filter Electronics category sales
* Find high-value orders based on revenue threshold

### Feature Engineering

* Create Revenue column using:

Revenue = Quantity × Price

### Data Aggregation

* Calculate total revenue by city
* Generate business insights using GroupBy operations

### Data Integration

* Merge sales data with customer information
* Perform customer-level analysis
